Le Dashboard vous permet d'analyser vos résultats.

- KPIs du Pipeline
- Répartition par statut
- Classement par niche / utilisateur
- Tâches terminées
- Nouveaux leads contactés

Vous pouvez filtrer ces statistiques par niche et/ou par utilisateur pour affiner les résultats.
