## API Publique

L’API Publique de Fructu permet aux utilisateurs premium du CRM d’interagir de manière programmatique avec leurs données. Vous pouvez notamment créer des contacts, consulter des informations, ou automatiser des actions directement depuis vos propres outils.

Conçue pour être simple, sécurisée et performante, cette API publique vous offre une flexibilité maximale pour connecter Fructu à votre écosystème logiciel (scripts internes, outils no-code, automatisations, etc.).

<Note>
  L'utilisation de l'API est réservé aux utilisateurs disposant d’un abonnement premium.
</Note>
