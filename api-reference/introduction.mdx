## Comment s'authentifier pour utiliser l'API ?

Tous les endpoints sont sécurisés grâce à une clé API propre à votre Workspace.

Insérez votre clé API dans le header de votre requête comme dans l'exemple ci-dessous.

```json
"Authorization": "Bearer {API_KEY}"
```

## Où trouver ma clé API ?

Chaque Workspace possède sa propre clé API qui lui permet d'intéragir avec l'API.

Vous trouverez votre clé API dans les paramètre de votre Workspace, dans l'onglet "API".

<Info>
  Vous devez être abonné à Fructu pour utiliser notre API publique
</Info>
